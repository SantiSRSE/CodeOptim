######################################################################
         Groupwise Registration Method for 2D Motion Estimation
######################################################################

This package implements with the Groupwise Registration Method for 2D Motion
 Estimation applied to segmentation propagation along the cardiac cycle 
for MR-CINE sequences.

It is available for download at <http://www.lpi.tel.uva.es>.

In order to use, please make sure to include the current folder and subfolders
 in MATLAB path. Use the provided demo script Groupwise_reg.m.
An example data file is also available for download at
<http://www.lpi.tel.uva.es>.
